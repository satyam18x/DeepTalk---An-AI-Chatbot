# DeepTalk - An AI Chatbot 

An AI-powered chatbot web application that allows users to interact with an intelligent assistant in real time.  
The chatbot uses Google's **Gemini API** for generating responses and is built with a modern frontend using **React** and **Tailwind CSS**.

---

## Overview

This project is a simple and responsive AI chatbot interface where users can send messages and receive intelligent responses generated by the Gemini AI model.  
The application focuses on providing a clean UI and smooth chat experience.

---

## Features

- Real-time AI chatbot interaction
- Clean and modern chat interface
- Fast responses using Gemini AI
- Responsive UI for desktop and mobile
- Styled with Tailwind CSS
- Dynamic message rendering

---

### Frontend
